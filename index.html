개인정보처리방침

Nailva — 네일샵 매니저 · com.wowpia.nailshopmanager · 최종 업데이트: 2026-06-24 · 시행일: 2026-06-24
요약. Nailva 는 기본적으로 모든 고객 데이터를 기기 내에 로컬로 저장합니다. 클라우드 백업을 켜거나 OAuth 로그인을 사용하면 일부 데이터가 당사 서버로 전송됩니다. 또한 앱 운영·개선·광고를 위해 익명 진단·크래시 로그·광고 ID 가 수집됩니다.

1. 운영자

운영자: Info-Gate (wowpia)
문의 이메일: startline2011@gmail.com

2. 수집·공유하는 데이터

아래 표는 본 앱의 Google Play 데이터 보안 선언과 동일합니다.

2.1 기기 내 로컬 저장 (수집 아님)

아래 데이터는 사용자가 직접 입력하며 기기 내 SQLite 데이터베이스에만 저장됩니다. 클라우드 백업을 활성화하지 않는 한 당사 서버로 전송되지 않습니다 (2.2 절 참고).

  • 고객 이름·전화번호·메모
  • 시술 기록 및 사진
  • 예약 정보
  • 매출·통계
  • 앱 설정 (언어·테마)

2.2 클라우드 백업 (선택, 옵트인)

설정 → 클라우드 백업에서 Google 또는 Kakao OAuth 로 로그인하면 아래 데이터가 당사 Nailoop 백업 서버로 전송·저장됩니다:

  • 이메일 주소, 닉네임, 소셜 계정 식별자 (Google ID / Kakao ID)
  • 서버 발급 사용자 ID 및 인증 토큰
  • 고객 이름·전화번호·메모
  • 시술 기록·사진
  • 예약 정보

언제든 클라우드 백업에서 로그아웃하여 전송을 중단할 수 있습니다. 서버에 이미 저장된 데이터를 삭제하려면 5 절을 참고하세요.

2.3 항상 수집 (자동)

데이터 유형제3자 서비스목적공유 여부
광고 ID, 기기 식별자Google AdMob광고 노출예 (Google 과 공유)
App Instance ID, 기기 정보Firebase Analytics (Google)분석아니요 (처리자)
Installation ID, 크래시 로그Firebase Crashlytics (Google)크래시 리포트아니요 (처리자)
익명 distinct ID, 앱 상호작용PostHog제품 분석아니요 (처리자)
기기 식별자, 크래시 컨텍스트Sentry에러 추적아니요 (처리자)

위 데이터는 클라우드 백업 사용 여부와 무관하게 수집됩니다. 앱 내에서 개별 옵트아웃은 제공하지 않지만, 기기 설정에서 광고 ID 재설정 또는 광고 동의 배너에서 동의 철회가 가능합니다.

3. 처리 목적

  • 앱 기능: 고객·예약·사진·매출 관리
  • 계정 관리: 클라우드 백업·OAuth 로그인 시 본인 식별
  • 분석: 기능 사용 측정을 통한 앱 개선 (PostHog, Firebase Analytics)
  • 진단·크래시 리포트: 버그 탐지 및 수정 (Crashlytics, Sentry)
  • 광고: 무료 기능 유지를 위한 광고 노출 (AdMob)

4. 제3자 제공

당사는 개인정보를 판매하지 않습니다. 자체 목적으로 데이터를 활용하는 유일한 제3자는 Google (AdMob)으로, 광고 ID 를 광고 노출에 사용합니다. 그 외 제3자 서비스(Firebase, PostHog, Sentry)는 모두 데이터 처리 계약에 따른 처리자로서 당사 지시에 따라서만 데이터를 처리합니다.

5. 데이터 보관 및 삭제

  • 로컬 데이터: 앱 내 삭제 또는 앱 제거 시까지 보관
  • 클라우드 백업 서버 데이터: 이메일로 삭제 요청 시까지 보관. 계정 삭제 안내 참고
  • Sentry 크래시 로그: 90 일 후 자동 파기
  • PostHog 분석 데이터: 7 일 후 자동 파기
  • Firebase Analytics·Crashlytics: Google 정책에 따라 보관
  • 익명화된 집계 통계: 개인 식별 불가 형태로 무기한 보관 가능

계정 삭제 절차: https://info-gate.github.io/nailva/delete-account/

6. 데이터 보안

  • 모든 데이터 전송은 HTTPS / TLS 로 암호화
  • 로컬 데이터는 앱 전용 샌드박스에 저장
  • 서버 데이터는 접근 제어된 Info-Gate 운영 인프라에 저장

7. 이용자의 권리

이용자는 다음 권리를 행사할 수 있습니다:

  • 앱 내 본인 데이터 열람
  • 앱 내에서 데이터 수정
  • 앱 내에서 개별 기록 삭제 또는 서버 전체 데이터 삭제 요청
  • 클라우드 백업 로그아웃을 통한 동의 철회
  • 기기 설정 → 개인정보 → 광고에서 광고 ID 재설정

8. 사용 권한

  • 카메라: 시술 사진 촬영 (로컬 저장; 클라우드 백업 활성화 시에만 서버 업로드)
  • 연락처 (READ_CONTACTS): 기기 주소록에서 고객 정보 가져오기 (선택)
  • 알림 (POST_NOTIFICATIONS): 예약 알림
  • 정확한 알람: 정시 예약 알림 예약
  • 인터넷: 광고·분석·OAuth 로그인·클라우드 백업

9. 아동 개인정보 보호

Nailva 는 네일샵 사장님(비즈니스 사용자)을 위한 앱으로 만 13 세 미만 아동을 대상으로 하지 않으며, 의도적으로 아동의 정보를 수집하지 않습니다.

10. 방침 변경

본 개인정보처리방침은 변경될 수 있으며, 중대한 변경 사항은 앱 내 공지 또는 앱 업데이트를 통해 안내됩니다. 상단의 "최종 업데이트" 일자는 최신 개정일을 의미합니다.
